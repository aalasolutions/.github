## Open Source

### [RAAG](https://github.com/aalasolutions/raag) · Python
**Relationship-Aware Augmented Generation** — a document intelligence layer for RAG pipelines. RAAG ingests documents, derives cross-document relationships from structure and explicit patterns, and guides retrieval using a document relationship graph. No LLM required for extraction. Structure-derived, deterministic, production-ready.

### [nginx-proxy-manager-sdk](https://github.com/aalasolutions/nginx-proxy-manager-sdk) · TypeScript
A zero-dependency TypeScript SDK for the Nginx Proxy Manager API. Programmatic proxy host management, Let's Encrypt SSL provisioning, DNS challenge support, and domain verification — fully typed for Node.js 18+.

### [Jazari](https://github.com/aalasolutions/vscode-jazari) · VS Code Theme
A dark and light VS Code theme with teal accents, rounded floating panels, and backdrop blur. Named after **Al-Jazari** (1136–1206), the Muslim polymath who built the world's first programmable machines. Available on the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=aala.jazari).

### [aala-review](https://github.com/aalasolutions/aala-review) · AI Skills
Installable code review skills for AI coding agents (Claude Code, Cursor, Codex, Continue). Covers JavaScript, TypeScript, Go, Python, PHP, Rust, HTML, and CSS/SCSS with framework overlays for React, Next.js, NestJS, FastAPI, Laravel, and more.
